# Riskflow Obsidian Vault

Open this folder as the Obsidian vault:

```text
/Users/Shared/Riskflow/obsidian
```

Use this vault as the research memory layer for Riskflow.

## Folders

- `concepts/` contains definitions and durable research concepts.
- `reports/` contains generated scan and event-study markdown reports.
- `assets/` can hold screenshots, diagrams, and supporting images.

Python calculates signals, baskets, compression, states, scores, and event studies. Obsidian explains, links, and remembers the research.

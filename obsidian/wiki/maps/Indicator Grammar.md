## Principle

Patterns are combinations of location, structure, trigger, confirmation, failure, and context tags.

## Next Memory Order

1. Preserve the user's visual read as an observation.
2. Tag it with grammar primitives.
3. Link it to similar hits, failures, and misses.
4. Convert repeated primitives into measurable events.
5. Test with forward relative-return evidence.
6. Promote only the simplest visual layer that improves readability and evidence.
